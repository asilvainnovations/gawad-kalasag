# Gawad Kalasag DRRM Excellence Web Application

## Overview
This project is a professional, production-ready static web application that presents an overview of the **Gawad Kalasag** awards of the Philippines. It highlights the purpose, significance, criteria, and scoring system used by the National Disaster Risk Reduction and Management Council (NDRRMC) to recognize excellence in Disaster Risk Reduction and Management (DRRM) and humanitarian assistance.

The application is designed with consistency in layout, style, and user interface, using standard web technologies.

## Features
- Structured presentation of Gawad Kalasag overview
- Clear listing of significance criteria
- Tabular display of scoring dimensions and weights
- Responsive and clean user interface
- Data-driven rendering using JSON

## Technologies Used
- HTML5 for structure
- CSS3 for styling and layout
- JavaScript for dynamic content rendering
- JSON for structured data storage

## Project Structure
The project follows a simple and maintainable file structure separating concerns for styling, logic, and data.

## How to Run
1. Download or clone the project files.
2. Ensure all files and folders remain in the same structure.
3. Open `index.html` in any modern web browser.
4. No server or build tools are required.

## Customization
- Update textual content by editing `assets/js/data.json`.
- Modify layout and visual design in `assets/css/styles.css`.
- Extend functionality or logic in `assets/js/main.js`.

## Intended Use
- Educational reference on Gawad Kalasag
- Demonstration of DRRM evaluation frameworks
- Template for DRRM-related award or assessment portals
- Informational site for LGUs, NGOs, private sector, and researchers

## Compliance
The scoring dimensions and descriptions align with the principles of:
- Republic Act 10121 (Philippine DRRM Act)
- National DRRM Framework and Plans
